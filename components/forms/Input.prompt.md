Labelled text field with red focus ring, hint and error states.

```jsx
<Input label="E-Mail" type="email" placeholder="name@beispiel.de" hint="Wir melden uns innerhalb von 24 Stunden." />
<Input label="PLZ" error="Bitte geben Sie eine gültige PLZ ein." />
```

Props: `label`, `hint`, `error`, `iconLeft`, plus all native input attrs. Focus shows a red glow ring.
