Bold signal-red price lockup, set the way Rehmann's promo materials do it.

```jsx
<PriceTag prefix="Voll-Service-Preis" value={2499} was={3299} note="inkl. Lieferung & Montage" size="lg" />
<PriceTag value="ab 1.111,–" size="md" />
```

Props: `value` (number → de-DE formatted, or string), `was` (struck previous price), `prefix` (eyebrow), `note` (caption), `size`, `currency` (default €), `align`. Extrabold, tabular figures, red.
