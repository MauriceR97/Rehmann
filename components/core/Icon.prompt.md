Lucide icon wrapper (the documented brand icon substitution). Inherits color from `currentColor`.

```jsx
<Icon name="shopping-cart" size={20} />
<Icon name="phone" size={18} color="var(--reh-red)" />
```

Requires `<script src="https://unpkg.com/lucide@latest"></script>` on the page. Use stroke icons only; size 18/20/24 in UI.
